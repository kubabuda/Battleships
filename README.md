# Battleships

Console single side game of Battleships

![](doc/Screenshot.png)

## Running

### Prerequisites
 - .NET Core 2.2
 - optional Visual Studio Code or Visual Studio for development and debugging

### Running console app
`dotnet run --project Battleships/Battleships.csproj`

### Running tests
`dotnet test`
